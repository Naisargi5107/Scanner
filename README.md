# QR Code Generator

A lightweight Python application that generates QR codes from URLs, text, or any custom data and saves them as image files. This project demonstrates the use of the qrcode library and image processing in Python.

# Overview

QR codes are widely used for sharing links, contact information, and other digital content. This project allows users to generate QR codes quickly with just a few lines of Python code.

# Features
Generate QR codes from URLs or text
Save QR codes as PNG images
Simple and beginner-friendly implementation
Fast and lightweight execution
Easy to customize and extend
Tech Stack
Python 3
qrcode
Pillow (PIL)
